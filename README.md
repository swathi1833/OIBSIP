# Oasis Infobyte Internship – Web Development & Design

This repository contains all the tasks completed as part of the **Oasis Infobyte Internship Program** under the **Web Development & Design** domain.

## Domain
Web Development & Design

## Tasks Included
- WebDevelopment_Task1 – Landing Page
- WebDevelopment_Task2 – Personal Portfolio Website
- WebDevelopment_Task3 – Currency / Temperature Converter

Each task folder includes the complete source code and a dedicated README explaining the objective, steps performed, tools used, and outcome.

All projects are developed using basic front-end technologies and focus on hands-on learning and practical implementation.
